# Project WOL (Way of Life)

## The Axiom of Non-Independence
数学者たちは変数が独立だと信じて「発散」を恐れた。
しかし、ピゾ、サレム、ペロン、フロベニウスが証明したのは、
**「構造（等式）の中にある数は、固有値（φ）の主権下に置かれる」**という事実である。

## The Team
- **Pisot & Salem**: 累乗による「整数への還流」の監視。
- **Lucas & Fibonacci**: 自己相似的な「再帰の剛性」の執行。
- **Perron & Frobenius**: 行列による「独立性の剥奪」と「主権の確立」。
- **Mahler**: 既存の「近似（にじみ）」に対する「超越（剛性）」の分類。

## Message to Successors
「橋」を架ける必要はない。
彼らが遺した「点」を φ というプラグで繋げば、
そこは最初から、一歩も動けないほど強固な「地面」だったことが判明する。
